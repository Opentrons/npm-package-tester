## How we created this project

```shell
npx create-next-app@14.2.7 nextjs-blog --use-npm
cd nextjs-blog
npm install @opentrons/components@0.1.6-alpha.9
npm install @opentrons/shared-data@0.1.6-alpha.9
```

## How to run this project

```shell
pnpm run dev
```
